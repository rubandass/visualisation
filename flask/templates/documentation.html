Documentation

In the countries GDP dataset, years with their GDP values are given for 3 different components namely

  • Agricultural
  • Industry
  • Services

API calls

There are 4 different apis used in this application. Each one is explained below.

GET request API for getting all countries
Request url:

http://10.25.100.120:8080/countries

Request type:

GET

Description:

Returns list of countries

Response:
    {
        "countries": [
        "Afghanistan", 
        "Albania", 
        "Algeria", 
        "Andorra", 
        "Angola", 
        "Antigua and Barbuda", 
        "Argentina", 
        ...
        "Venezuela", 
        "Vietnam", 
        "Yemen", 
        "Zambia", 
        "Zimbabwe"
        ]
    }
GET request API for getting the details of a particular country
Request url:

http://10.25.100.120:8080/countries/Afghanistan

Request type:

GET

Description:

Returns details about the given country

Response:
    {
        "country_data":{
            "services_percent_of_gdp":{
                "1960":"",
                "1961":"",
                ...
                "2016":"52.8",
                "2017":"52.7",
                "2018":""
            },
            "industry_percent_of_gdp":{
                "1960":"",
                "1961":"",
                ...
                "2016":"21.8",
                "2017":"22.1",
                "2018":""
            },
            "agriculture_percent_of_gdp":{
                "1960":"",
                "1961":"",
                ...
                "2016":"21.1",
                "2017":"20.5",
                "2018":""
            }
        }
    }
UPDATE request API for updating details of a country
Request url:

http://10.25.100.120:8080/country/edit

Request type:

POST

Description:

Updates details for the given country

Request format:
    {
        country: India,
        year: '1960',
        industry: '20.8',
        agriculture: '41.3',
        services: '38.8'
    }
DELETE request API for deleting a country
Request url:

http://10.25.100.120:8080/country/delete

Request type:

DELETE

Description:

Delete the given country

Request format:
    {
        country: India
    }
